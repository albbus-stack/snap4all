How it works

To finalize the installation of Snap4all the device must be restarted. Once restarted, the new features installed will be shown in this menu. Without restarting you can use the Termux Console or read the Information. -> If the Termux Console is disabled please check your internet connection, then close and reopen the app before restart.
  • If you want to open the termux console simply click on 'Termux Console'.
  • If you want to open IOT APP EDITOR, when it is enabled, simply click on 'IOT APP EDITOR'.
  • If you want to open the IOT APP DASHBOARD, when it is enabled, simply click on 'DASHBOARD'.

The IOT APP EDITOR and DASHBOARD buttons are enabled and visible only after the Node-RED server starts up.

Example of use

  1. Install the application.
  2. Reboot your phone.
  3. Open the application.
  4. Click on IOT APP EDITOR.
